About This Project

This project was created for AP CS Principles. It simulates a game of Rock Paper Scissors, with multiple variations allowing for player choice. The user can choose between classic RPS, and an Avatar mode that includes additional elements like Water, Fire, and Earth. The game keeps track of scores and provides an engaging experience against a computer opponent.


Technologies Used: This project was created in XCode, using Swift. It was created for the iPhone.


Key Features:

  • Avatar Mode: A novel way to experience RPS! Allows the user to play using elements.
  • Users can choose between best of 3 and first to 5
  • A help button is included for both modes to assist players (especially useful for avatar mode).
