About This Project

This project was created last year for AP Computer Science Principles. It simulates a game of blackjack, allowing the user to experience the thrill of the game against a computer opponent.


Technologies Used: This project was created in XCode, using Swift. It was created for the iPhone.


Key Features:

  • Users can hit or stand depending on their set of cards.
  • Players compete against a CPU
  • The app tracks scores for both the player and the house
